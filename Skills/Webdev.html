Mes Compétences - Développement Web

En tant que développeur passionné, j'ai acquis des compétences solides dans les technologies clés du développement web, que ce soit côté frontend ou backend.

Compétences Frontend :

  • HTML5 : Maîtrise des balises et des normes HTML5 pour la création de structures web.
  • CSS3 : Expérience approfondie dans la création de mises en page attrayantes et réactives avec CSS3.
  • JavaScript : Compétence dans l'utilisation de JavaScript pour rendre les sites web interactifs et dynamiques.
  • Frameworks : Familiarité avec l'utilisation de frameworks tels que Bootstrap pour accélérer le développement frontend.

Compétences Backend :

  • Node.js : Expertise dans le développement backend avec Node.js, incluant la création d'APIs REST avec Express. Tests réalisés avec Postman.
  • Python : Maîtrise de Python pour le développement backend, notamment avec `discord.py` pour des projets de bots et l'automatisation de tâches.
  • MySQL : Expérience dans l'utilisation de MySQL pour la gestion et la manipulation de bases de données relationnelles.
  • SQL : Connaissance des requêtes SQL pour l'interrogation et la gestion de bases de données.

Projets Personnels :

  • Bot Discord : Développement d'un bot Discord en `discord.py` pour automatiser la gestion de serveurs, modération et fourniture d'informations en temps réel.
  • Application de gestion de tâches : Création d'une application en Node.js pour organiser et suivre des tâches avec une interface utilisateur intuitive.
  • Portfolio en ligne : Développement d'un site personnel en Node.js et Express pour présenter mes projets et compétences.
  • Projet de base de données MySQL : Conception et implémentation d'une base de données relationnelle pour un projet de gestion de contenu, avec des interfaces utilisateur et des requêtes SQL optimisées.

En plus de ces compétences spécifiques, je suis constamment engagé dans l'apprentissage de nouvelles technologies et dans l'amélioration de mes compétences pour créer des sites ou applications modernes et performants.
